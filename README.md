# Database Design & Relational Modeling

This repository contains my work related to **database design, relational modeling, and ER diagram development**.

The goal of this repository is to practice and improve my understanding of **relational databases**, including schema design, entity–relationship modeling, normalization, and database structure planning.

The projects included here focus on modeling real-world systems and transforming requirements into well-structured database schemas.

## Topics Covered

- Entity–Relationship (ER) diagrams
- Relational database schema design
- Primary keys and foreign keys
- One-to-many and many-to-many relationships
- Database normalization
- Real-world database modeling

## Tools

The diagrams and models in this repository were created using database modeling tools such as:

- Oracle SQL Developer Data Modeler
- ER diagram design tools

## Purpose

This repository serves as a **learning portfolio** documenting my progress in database design and relational database systems.

It will continue to grow as I practice designing database schemas for different types of systems and applications.
